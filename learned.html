Key Takeaways & Technical Growth

My internship provided a comprehensive overview of the enterprise security landscape. From theoretical frameworks to the hands-on management of defense tools, here is a summary of the core concepts and technologies I mastered.

1. Core Security Concepts

  • The CIA Triad: Confidentiality, Integrity, and Authentication.
  • The Security Trade-off: Balancing security and convenience.
  • Cyber Attack Types: Malware, Ransomware, DoS, SQL Injection, Cryptojacking.

2. The Security Tech Stack

  • Qualys: Vulnerability Management, Asset Management, Threat Intelligence.
  • Symantec DLP: Endpoint monitoring for sensitive data transfer and unauthorized AI usage.
  • Trend Micro: Email and endpoint security (XDR), sandboxing, domain look-alike detection.
  • KnowBe4: Security awareness training and simulated phishing campaigns.

3. Vulnerability Assessment & Penetration Testing (VAPT)

  • VA vs PT: Identifying flaws vs exploiting them.
  • Testing Methodologies: White Box, Black Box, Grey Box.
  • Reporting: CVEs, risk ratings, remediations using Agile workflows.

4. Email Security & Phishing

  • Email classification: Clean, Spam, Threat.
  • Sender analysis, domains, grammar, attachment hashing.
  • Strategic insight: Most effective phishing times (Tue–Wed, 9–11 AM).

5. Cryptography

  • Symmetric, Asymmetric encryption, and Hashing.
  • Applied concepts through the custom Piano Cypher.
